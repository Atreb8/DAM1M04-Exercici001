Benvinguts a CSS bàsic

CSS permet controlar colors, tipografies, mides, marges, espais i fons. Això fa que les pàgines siguin atractives i professionals millorant l'experiència de l'usuari. Sense CSS les pàgines es veurien amb l'estil predeterminat del navegador que és molt bàsic i poc visual.

CSS és un estàndard de la W3C àmpliament suportat per tots els navegadors. Aprendre CSS és indispensable per qualsevol desenvolupador web ja que sense CSS les pàgines serien funcionals però poc atractives

  • Separació de contingut i presentació
  • Responsivitat i adaptabilitat
  • Millora del rendiment i la consistència
  • Interactivitat visual
